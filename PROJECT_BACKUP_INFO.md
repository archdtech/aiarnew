   ```bash

   ```

3. **Navigate to project directory**
   ```bash
   cd arabic-tech-news-summarizer
   ```

4. **Install dependencies**
   ```bash
   npm install
   ```

5. **Set up database**
   ```bash
   npm run db:push
   ```

6. **Start development server**
   ```bash
   npm run dev
   ```

7. **Access the application**
